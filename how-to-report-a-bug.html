# How to Report a Bug Contrary to popular belief developers love fixing bugs. What developers hate is searching for bugs. Poor bug reports can waste a lot of time and money by sending developers on a wild goose chase. Good bug reports help locate problems so they can be fixed. Follow these guidelines to create a useful bug report. ## What are trying to do? Describe what you're trying to accomplish with the software. ## How does the bug appear? What steps need to be taken to make the bug happen? ## Done Congratulations! Your bug report is now above average.
